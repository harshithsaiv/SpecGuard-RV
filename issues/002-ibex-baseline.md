## Build vanilla Ibex Verilator baseline

Labels: `rtl`, `ibex`, `verilator`

Clone lowRISC Ibex and build the `simple_system` Verilator example.

- Clone Ibex
- Build `simple_system` Verilator target
- Run `hello_test` (or equivalent baseline program)
- Document build commands in `docs/ibex-baseline.md`

- Vanilla Ibex simulation runs successfully
